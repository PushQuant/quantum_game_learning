# quantum_game_learning
Have a game based on quantum computing to learn about web-dev and also q-computing

Level 1 - Superposition

You start with a Qubit, which is unobserved and have the option to observe it or not. When observing it, it has a random chance to go to either the 0 state or the 1 state.

Level 2 - Quantum Entanglement

There are two floating Qubits, which are entangled through a vibrating/pulsating line. You can either measure Qubit A or B. Both will react and get into the same state.
